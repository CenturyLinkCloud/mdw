Swagger Validator

The Swagger Validator activity checks a process's request for conformance with the requirements specified in its Swagger schema. The results are accumulated in a Limberest Result object. Default handling of the result is as follows (override handleResponse() to customize):

  • Success: returns an activity result of true
  • Failure: sets the response payload and HTTP status code according to the result, and returns an activity result of false

    • Configuring

    Validation is applied to the selected request items.

    Strict means unexpected elements cause validation failure (vs ignored).
